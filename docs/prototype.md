# 产品原型 - 阅卷工作台型

## 视觉方向

轻量现代 SaaS 风：浅色背景、稳定左侧导航、清晰数据层级、少阴影、低装饰。首屏服务教师的日常工作，而不是营销展示。

## Web 教师端

### 工作台

- 顶部显示今日待处理：待批试卷、待复核主观题、未提交作业、薄弱知识点。
- 中部左侧为扫描队列，展示处理状态。
- 中部右侧为主观题复核入口，突出 AI 建议分和教师待确认数量。
- 底部显示知识点薄弱排行和班级作业完成情况。

### 主观题批阅

核心交互为左右分屏：

- 左侧：标准答案、评分规则、知识点、满分。
- 右侧：学生试卷识别结果、学生答案、AI 建议分、AI 理由。
- 中间/底部：教师修改分数、接受 AI 建议、驳回、保存并进入下一题。

教师拥有最终裁决权，AI 只提供建议。

### 试卷模板制作

- 中间显示试卷页面。
- 支持拖拽矩形框选题目区域。
- 右侧设置题号、题型、分值、标准答案、知识点。
- 保留 AI 拆卷建议入口，但第一版以教师确认为准。

### 学情分析

- 试卷整体：平均分、最高分、最低分、分数段。
- 题目维度：正确率、难题识别。
- 知识点维度：掌握度排行。
- 学生维度：错题分布、连续未提交预警。

## Android 学生/家长端

- 任务列表：待完成、已提交、已批改。
- 上传作业：拍照/相册/PDF 上传，提交状态。
- 错题本：原题、我的答案、正确答案、知识点、再练入口。
